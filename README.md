reactive-mongo
=================

Mechanisms to connect to a MongoDB database (reactive mongo implementation)

To utilise this library add this to your sbt build file

```
"com.cjww-dev.libs" % "reactive-mongo_2.11" % "<SEE_VERSION_ABOVE>" 
```
